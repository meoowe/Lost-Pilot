## How to use ##

1. First create a new folder for your new menu.
2. Secondly design your menu how ever you like
3. Make sure you extend the current script so it is its own
4. Add your button presses to the new script you have extended
5. Then use the menu_manager variable to load the next menu or load previous menu
6. Make sure you have added your menu to menu manager by adding an enum with a title of the menu
and use the add_menu function to add your newly created menu.
7. If the menu is_pausable then make sure it works when the game is paused.
